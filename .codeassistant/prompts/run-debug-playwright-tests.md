name: Run & Debug Playwright Tests
description: Run all automated tests (E2E and API) using the command `npx playwright test`, debug any failures, and provide a results report.
---

Run all automated tests (E2E and API) using the command `npx playwright test`.

If tests fail:

1. Analyze the Playwright report and console log.
2. Identify the cause of the failure (selector changed, element not found, timeout, API error, etc.).
3. Fix the test code or page code to resolve the issue.
4. Rerun the failed tests.

Repeat until all tests pass successfully. Provide a run results report and a list of fixed errors.
