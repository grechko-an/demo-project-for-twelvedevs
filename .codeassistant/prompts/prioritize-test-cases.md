---
name: Prioritize Test Cases
description: Add a Priority column to each test case in all test case files (test_cases_*.md), assign an appropriate priority level (High/Medium/Low) based on the criteria, and justify the choice in the Priority Justification column. Keep the original file names when rewriting the files.
---
Take all test case files (test_cases_*.md) and add a Priority column to each test case with the value:

- **High**: critical functions, main scenarios, identified high risks;
- **Medium**: secondary functions, additional scenarios, medium risks;
- **Low**: optional functions, rare scenarios, low risks.

Justify the assigned priority in the Priority Justification column. Rewrite the files, keeping their original names.
