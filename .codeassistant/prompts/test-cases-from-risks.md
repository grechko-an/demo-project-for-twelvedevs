---
name: Test Cases from Risks
description: Generate test cases to verify identified vulnerabilities and weak spots based on the risk analysis report.
---

Based on the risk analysis report (provided earlier), generate test cases to verify the identified vulnerabilities and weak spots. Create 1–3 test cases for each risk.

Output format: a separate file `test_cases_risks.md`. Each test case must include:

- **ID:** unique number;
- **Title:** brief description;
- **Precondition:** what needs to be done before the test;
- **Steps:** step‑by‑step description of actions;
- **Expected Result:** what should happen;
- **Related Risk:** link to the risk ID from the report.

Group test cases by risk severity level.
